# NORE — design-art-gallery-bluechip

> **"Only what cannot be sold by description."**

A design study exploring a fictional blue-chip art gallery — Gagosian / Pace / Hauser & Wirth tier — targeting **collectors aged 40–60 with $5M+ investable assets** who attend Art Basel and Frieze, hold TEFAF memberships, and never see a price tag in person.

NORE is a fictional gallery created for this design study. It is not a real gallery, dealer, artist, or work.

## Overview

| | |
|---|---|
| **Brand** | NORE |
| **Persona** | bluechip |
| **Live Site (planned)** | `design.art-gallery-bluechip.riumu.net` |

## Design Concept

- **Color**: Pure museum white `#FFFFFF` × ink black `#1A1A1A` × deep oxblood `#5C1A1B` (accent only)
- **Typography**: Bodoni Moda (display, Didone) × Inter (body)
- **Aesthetic**: Museum-grade white cube × single-work hero × scroll-driven zoom-out
- **UX**: Five-item nav, no cart, no price, no flashy motion. Enquire is a separate page — no modal.
- **Caption style**: Side-rotated 90° museum-label format.

## Tech Stack

- Pure HTML + CSS Custom Properties + Vanilla JS
- Google Fonts CDN (Bodoni Moda + Inter)
- No framework, no build step — GitHub Pages ready
